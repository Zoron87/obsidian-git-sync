Как выбирать тип параметра метода

Вот практическая таблица.

|Нужно внутри метода|Что принимать|
|---|---|
|Один раз пройтись по элементам|`IEnumerable<T>`|
|Нужен `Count`|`IReadOnlyCollection<T>`|
|Нужен `Count` и доступ по индексу|`IReadOnlyList<T>`|
|Нужно добавлять/удалять элементы|`ICollection<T>`|
|Нужно менять по индексу, вставлять, удалять по индексу|`IList<T>`|
|Нужно читать по ключу|`IReadOnlyDictionary<TKey, TValue>`|
|Нужно менять словарь|`IDictionary<TKey, TValue>`|
|Нужно проверять уникальность/наличие|`IReadOnlySet<T>`|
|Нужно менять множество|`ISet<T>`|
|Нужен EF-запрос и построение SQL|`IQueryable<T>` — осторожно|
