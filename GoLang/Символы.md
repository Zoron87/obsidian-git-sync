var c rune = '#'
```go
func main() { 
	var char rune 
	fmt.Scanf("%c", &char) 
	fmt.Println("Вы ввели:", string(char)) }
```

`fmt.Scanf` — это функция, которая позволяет считывать ввод с консоли по определённому формату. Если простыми словами, ты заранее говоришь программе, что именно ты хочешь прочитать: символ, число, строку и так далее.

**`"%c"`** — это формат, который говорит функции: "Мы хотим прочитать один символ". Буква `c` comes от слова "character" (символ).

#### Способ 2: Чтение одного символа через `bufio`

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    reader := bufio.NewReader(os.Stdin)
    char, _, _ := reader.ReadRune()
    fmt.Println("Вы ввели:", string(char))
}
```
1. `bufio.NewReader(os.Stdin)` - создает буферизованный читатель для консоли
2. `reader.ReadRune()` - считывает **один символ** (даже если это эмодзи или кириллица)
3. `string(char)` - преобразует код символа обратно в читаемый вид
### Представление символов в компьютере
```go
import "fmt" 
func main() { 
	c := '#' 
	code := int(c) 
	fmt.Println(code) // 35 }
```

### Сравнение символов

Символы можно сравнивать — сравниваются их **коды**:

```go
package main

import "fmt"

func main() {
    first := 'B'
    second := 'v'
    fmt.Println(first < second) // true
}
```

Здесь `'B'` действительно меньше `'v'`, так как имеет меньший код.

Проверка, **является ли символ заглавной латинской буквой**:

```go
package main

import "fmt"

func main() {
    var c rune = 'B'
    if c >= 'A' && c <= 'Z' {
        fmt.Println("Заглавная буква английского алфавита")
    }
}
```
