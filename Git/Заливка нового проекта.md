Перед заливом на git нового проекта нужно сделать подготовительные шаги:

1. Откройте терминал в папке проекта
```bash
cd C:\Users\repos\MyProject
```

2. Инициализировать git
   ```bash
   git init
   ```

3. Добавь `.gitignore`
   ```bash
   dotnet new gitignore
   ```

4. Добавь файлы и сделай commit
   ```bash
   git add .
   git commit -m "Initial commit: MyProject"
   ```

5. Привяжи Git репозиторий
   ```bash
   https://github.com/YourOrg/YourProject/_git/MyProject
   ```
6. Заливаем проект
   ```bash
	git remote add origin https://github.com/YourOrg/YourProject/_git/MyProject
	git branch -M main
	git push -u origin main

   ```

Если ветка не пуста, то сначала может потребоваться выполнить
```bash
	git pull
```
